# Проверка скорости интернета

Скрипт последовательно скачивает указанный файл 10 раз, измеряет время каждого
запроса и выводит общий объем данных и среднюю скорость.

## Требования

- Python 3.8 или новее
- доступный по HTTP или HTTPS крупный файл

Дополнительные библиотеки не нужны.

## Запуск

```bash
python3 main.py https://example.com/large-image.jpg
```

Пример результата:

```text
Request  1: 1.213 s, 10.00 MB
...
Request 10: 1.185 s, 10.00 MB

Average request time: 1.201 s
Downloaded data: 100.00 MB
Average speed: 8.33 MB/s (66.61 Mbit/s)
```

Средняя скорость рассчитывается по суммарному объему данных и суммарному
времени всех запросов. Для более точного результата используйте крупный
статический файл на быстром сервере: измерение зависит не только от вашего
соединения, но и от скорости сервера.
